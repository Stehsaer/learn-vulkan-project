## 流水线的演变

### 经典绘制流水线

#### 设备特点

- **CPU**：顺序执行，按顺序逐个处理数据
- **GPU**：并行执行，一次性处理大量数据

#### 主要理念

将常用操作打包好，由CPU向GPU传输数据和指令，控制GPU处理数据

#### 发展历史

1. 固定流水线、顶点数组、显示列表、纹理对象、帧缓冲

   - **OpenGL 1.0** (1992)

   - **3dfx Glide** (1995)：“Voodoo显卡”
   - **DirectX 1.0** (1995)

2. 着色语言、像素着色器
   - OpenGL 2.0 >>> *GLSL* (2004)
   - DirectX >>> *HLSL* (2000)

3. 几何着色器

   - OpenGL 3.0 (2008)
   - DirectX 10 (2006)

   实现了模型LOD

4. 曲面细分

   - OpenGL 4.0 (2010)
   - DirectX 11 (2009)

   可产生曲线
